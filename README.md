Catalogo-EPS. README.

DESCRIPCION:

Este catalogo nace como solución a una problemática muy común en Colombia ligada a las EPS. Este catalogo cuenta con un CRUD completo integrado y dinámico. Al ejecutar el código, verificaras un login donde podrías iniciar sesión, crear una cuenta o modificar tu PIN. Después del login, se ubica la sección principal donde veras diferentes medicamentos en presentación card y su respectiva información(Nombre, imagen y stock). Al acceder a cualquier medicamento, veras su información detallada y dos opciones para solicitar dicho medicamento. Sin importar la elección, el sistema lanzara 4 fechas disponibles las cuales el usuario tendrá que elegir para solicitar su medicamento.

INSTALACION:

Descarga el archivo ZIP del proyecto Catalogo-EPS. Ingresa a la carpeta del proyecto y activa el entorno virtual si existe. Luego ejecuta Python app.py desde la carpeta principal. Abre en el navegador http://127.0.0.1:5000/ para acceder a la aplicación y navega por el diverso catalogo.
